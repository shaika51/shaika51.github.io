Course Description

This course introduces both fundamental and advanced techniques in web application programming and hacking. You will explore front-end development with HTML5, CSS, JavaScript, and back-end with PHP and MySQL, alongside learning about common web vulnerabilities and their defenses.

Topics Covered

  • HTML, CSS, JavaScript fundamentals
  • AJAX, jQuery, and RESTful APIs
  • PHP, MySQL, database connectivity
  • Web security: XSS, SQLi, CSRF, Session Hijacking
  • Secure coding principles
  • Git and GitHub deployment

Projects & Hackathons

  • Lab exercises (4)
  • Hackathon exercises (4)
  • Individual projects (2)
  • Team project & final presentation
